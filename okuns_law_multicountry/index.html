📋 研究全流程

📚 Stage 1: 文献调研与创意发现

WebSearch + scansci-pdf MCP + Skill: research-pipeline

通过 WebSearch 检索 Springer、ScienceDirect、MDPI、EconStor 等数据库。使用 research-pipeline Skill 中的文献调研流程发现 Neely (2010) 奥肯定律更新研究的缺口——缺乏后疫情时代的跨国比较证据。使用 scansci-pdf MCP 下载论文原文（Russnak 2023、Boda 2023 等）。

🔌 Stage 2: MCP 数据抓取

OpenEcon MCP + FRED MCP + World Bank Data360 MCP + OECD MCP + chrome-devtools MCP

通过统一 MCP 协议从 FRED 获取美国 GDPC1/UNRATE、加拿大统计局季度 GDP、欧盟统计局德国 GDP增长率与失业率。OpenEcon MCP 作为主数据通道，FRED MCP 作为验证通道。使用 chrome-devtools MCP 自动注册 FRED API Key。

🐍 Stage 3: Python 数据预处理

prepare_multicountry_data.py (Python 3.11)

数据清洗、加拿大年度失业率→季度线性插值（annual_to_quarterly函数）、德国GDP指数重建（从增长率反推2010Q1=100基准）、三国数据合并为统一面板CSV（194 obs × 7 columns）。

📈 Stage 4: Stata 计量分析

Stata MP 18 + stata-mcp (mcp-remote bridge) + Skill: auto-review-loop

分国 OLS 回归（4个子样本：全样本/排除COVID/COVID前/COVID后）、Newey-West HAC 稳健标准误（max lag=4）、混合固定效应模型（国家虚拟变量）、交互效应检验（i.cid#c.gdp_growth）、盈亏平衡增长率计算、3个图表导出。全程通过 stata-mcp 桥接（npx mcp-remote → http://localhost:4000/mcp-streamable → Stata MP 18 batch mode）。

🔄 Stage 5: 自动审稿循环

Skill: auto-review-loop + GPT-5.4 ×high

使用 auto-review-loop Skill 进行多轮学术审稿。每轮：GPT-5.4 xhigh 评审 → 输出评分(1-10) + 弱点列表 → 自动实现修改 → 重新评审。修复了文献覆盖不全面、统计表述不严谨、方法讨论深度不足等问题。

📄 Stage 6: 双语论文生成

PDFLaTeX + XeLaTeX + ctex + Skill: writing-plans

使用 writing-plans Skill 规划论文结构并输出英中双语的 LaTeX 源码。英文版使用 PDFLaTeX 编译（12页），中文版使用 XeLaTeX + ctex 宏包编译（11页，688KB）。含 JEL 分类、关键词、参考文献（BibTeX natbib格式）、附录数据说明。

📐 Stata 计量分析详解

完整的 296 行 do-file，覆盖从数据导入到图表导出的全部分析流程。

do-file 结构总览

/* okun_multicountry.do — 296 lines */

* 1. DATA IMPORT (lines 30–40)
import delimited using "okun_multicountry_data.csv"

* 2. DATA PREPARATION (lines 42–95)
* - Generate quarter date: yq(year, quarter)
* - Compute ΔU: d.unemp (quarterly change)
* - Compute annualized GDP growth: ((Yt/Yt-1)^4 - 1) × 100
* - Generate country ID: gen/replace (避免encode字母序BUG)
* - Generate contiguous time t2 = _n (for Newey-West)
* - Sub-sample markers: pre_covid, post_covid, excl_covid

* 3. COUNTRY LOOP (lines 97–192)
forvalues c = 1/3 {
  * A. Full sample OLS + robust SE
  * B. Excl. COVID OLS
  * C. Pre-COVID OLS
  * D. Post-COVID OLS
  * E. Newey-West Excl. COVID (tsset t2, lag(4))
  * F. Newey-West Full Sample
}

* 4. POOLED MODELS (lines 194–230)
* - Country FE: reg d_unemp gdp_growth cntry_1 cntry_2
* - Interaction: reg d_unemp gdp_growth i.cid#c.gdp_growth

* 5. SUMMARY TABLE (lines 232–268)
* - Collect coefficients, SE, t, R², break-even
* - Export formatted table to log

* 6. GRAPHS (lines 270–296)
* - Scatter plot by country + regression lines
* - GDP growth time series by country
* - Unemployment time series by country

核心回归代码

* 奥肯定律差分形式: ΔU_t = α + β × g_t + ε_t
* g_t = ((GDP_t / GDP_{t-1})^4 - 1) * 100 (年化季度增长率, %)

forvalues c = 1/3 {
  preserve
  keep if cid == `c'
  
  di "A. Full Sample (OLS):"
  reg d_unemp gdp_growth, robust
  
  di "B. Excl. COVID (OLS):"
  reg d_unemp gdp_growth if excl_covid == 1, robust
  
  di "E. Newey-West (Excl. COVID):"
  tsset t2 /* 连续时间变量 */
  newey d_unemp gdp_growth if excl_covid == 1, lag(4)
  
  restore
}

混合固定效应模型

* 国家虚拟变量 (避免d_unemp与d_*命名冲突！)
tab cid, gen(cntry_) // 使用cntry_前缀而非d_前缀

* 混合FE回归 (全样本)
reg d_unemp gdp_growth cntry_1 cntry_2, robust
* 结果: β = -0.0471, R² = 0.1974, N = 191

* 交互效应: 检验各国Okun系数是否相等
reg d_unemp gdp_growth i.cid#c.gdp_growth cntry_1 cntry_2, robust
* H0: β_USA = β_CAN = β_DEU
test 1.cid#c.gdp_growth = 2.cid#c.gdp_growth = 3.cid#c.gdp_growth

📉 Stata 踩坑记录

🐛 Bug 1: encode 字母序陷阱

encode country, gen(cid) 按字母序分配：CAN→1, DEU→2, USA→3。但标签定义为 1=USA, 2=CAN, 3=DEU，导致 cid==1 选择了加拿大而非美国，给出完全错误的 "美国" 结果（β=-0.008 而非 -0.19）。
修复：使用显式 gen/replace：

gen cid = .
replace cid = 1 if country == "USA"
replace cid = 2 if country == "CAN"
replace cid = 3 if country == "DEU"
label define cntrylbl 1 "USA" 2 "CAN" 3 "DEU"

🐛 Bug 2: d_* 通配符匹配到因变量 d_unemp

tab cid, gen(d_) 生成 d_1, d_2, d_3；回归式 reg d_unemp ... d_* 中 d_* 通配符匹配到 d_unemp 本身，导致完美多重共线性（R²=1.0）。
修复：用不冲突前缀 cntry_ 替代 d_，回归中显式列出虚拟变量名称。

🐛 Bug 3: COVID 剔除导致 Newey-West 时间不连续

剔除 2020Q2-2021Q1 后时间序列出现间隔，直接用 tsset quarter 导致 Newey-West 报错。
修复：生成连续时间变量 gen t2 = _n 作为 Newey-West 的时间索引。

📌 核心发现

  • 🇺🇸 美国：全样本 β=-0.1914，R²=0.7472，t=-13.54——看似完美的奥肯定律。但排除COVID后 β=-0.0202（不显著），COVID前 β=-0.0047（R²=0.003）——美国奥肯关系几乎完全由2020年疫情衰退的单一事件驱动。
  • 🇩🇪 德国：排除COVID后 β=-0.0471（t=-4.39），COVID前 β=-0.0409（t=-2.99），COVID后 β=-0.0491（t=-3.30）——三个子样本的系数高度一致（≈-0.044），体现了德国劳动力市场的结构稳定性。Kurzarbeit（短时工作制）使企业通过减少工时而非裁员应对冲击。
  • 🇨🇦 加拿大：全样本 β=-0.0084（显著），但数据经年→季插值处理，质量受限。COVID后关系增强（β=-0.0287，R²=0.382），但仅有19个观测值。
  • 混合FE：β=-0.0471（p=0.105），F检验拒绝系数相等假设，确认跨国产出-失业关系存在显著异质性。
